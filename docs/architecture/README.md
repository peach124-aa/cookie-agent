# Architecture Documentation (`docs/architecture/`)

This directory is designated for storing high-level architectural designs, system component diagrams, data flows, and physical process layouts for the Cookie Agent project.

---

## Guidelines

- **Focus**: High-level structural documentation only.
- **No Implementation**: Do not include actual Python source code, runtime integrations, or direct code implementations here.
- **Tools**: Design artifacts, mermaid diagrams, and logic schemas should be placed here to document how modules (Capture, Vision, State Tracker, Policy, Controller) interface with each other.
